(Recipes for) Resourcing the DOI

This page demos some simple use cases for managing resource collections with Digital Object Identifiers (see the DOI website for further info). The aim is to achieve consistently structured resource collections which will enable the wide deployment of a DOI Services economy. These examples conform to the YADS Model Schema.
