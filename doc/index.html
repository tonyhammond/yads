YADS - A Description Service

YADS implements a simple, safe and predictable recursive data model for describing resource collections. The aim is to assist in programming complex resource descriptions across multiple applications and to foster interoperability between them.

Background: This initiative originates from work within the scholarly publishing community in deploying the Digital Object Identifier (DOI) as a strategic resource to manage resource collections.
