yads: download (step 1)
This is yads, a tool which has been developed for the DOI Services project. It shows how DOIs as public identifiers can be resolved to multiple resources and how those resources are related.

Background documentation includes a schematic overview of the DObject Model, and some general Design Principles. The aim is to supply a framework for content providers to deploy strategic links over the Internet.
